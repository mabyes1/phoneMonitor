# BOOX Go Color 電子紙特化：產品事實

查核日期：2026-07-11

- BOOX Go Color 7 與 Go Color 7 (Gen II) 都使用 7 吋 E Ink Kaleido 3 彩色電子紙。
- 面板解析度為 1680 × 1264；黑白內容 300 PPI，彩色內容 150 PPI，最多 4096 色。
- 初代 Go Color 7 使用 Android 12；Go Color 7 (Gen II) 使用 Android 13。
- 電子紙不適合連續影像串流、漸層、半透明疊色與頻繁局部更新；本專案因此不在此模式載入虛擬顯示器串流，並將資訊板／額度輪詢降至 15／30 秒。
- UI 基準：淺色紙面、高對比深色文字、實線分隔、較粗字重、至少 44px 觸控目標，避免背景照片與動畫。

來源：BOOX 官方產品規格與官方說明中心。
