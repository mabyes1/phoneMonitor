遠端登入只接受 HTTPS。若還沒設定，請在 Host 啟動前設定 VIBEDECK_REMOTE_PASSWORD 或 PHONEMONITOR_REMOTE_PASSWORD。

iPhone 只用 HTTPS

長亮、主畫面 App、穩定連線都需要 HTTPS。這台 iPhone 不要再用 HTTP 開 VibeDeck。

  1. 先安裝本機 HTTPS 憑證（只做一次）
  2. 設定 → 一般 → 關於本機 → 憑證信任設定 → 開啟完整信任
  3. 回到這裡按「開啟 HTTPS」
1. 安裝 HTTPS 憑證 複製用的 HTTPS 網址

Android 瀏覽器可用 HTTP 測試；iPhone 固定走 HTTPS，不再切來切去。
